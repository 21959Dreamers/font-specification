### 字体风格和字重 | Font Styles and Weights

此字体包含以下风格：
- **Light、Regular、Medium、Bold、Heavy**
- **Text Regular、Text Medium**
- **Compact Light、Compact Regular、Compact Bold**
- **Wide Regular、Wide Bold**

每种风格提供独特的字重和设计，适用于不同的品牌和设计环境。
